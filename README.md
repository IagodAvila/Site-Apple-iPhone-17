📱 iPhone 17 Pro — Apple Website Clone

Um clone moderno da página do iPhone 17 Pro, inspirado no design minimalista e na experiência visual do site oficial da Apple.

Este projeto foi desenvolvido com React + Tailwind CSS, focando em componentização, responsividade e fidelidade visual ao estilo da Apple.

O objetivo foi praticar boas práticas de desenvolvimento front-end moderno e construção de interfaces elegantes.

🛠 Como executar o projeto

Clone o repositório:

git clone https://github.com/IagoDavila/Site-Apple-iPhone-17.git

Entre na pasta do projeto:

cd Site-Apple-iPhone-17

Instale as dependências:

npm install

Execute o projeto:

npm run dev
